<h1>SOLTEC 2 - Proyectos SPOS</h1>
<p>Bienvenido al repositorio <strong>SOLTEC_2</strong>, que contiene los proyectos principales del sistema SPOS de SOLTEC.</p>

<h2>Proyectos incluidos</h2>
<ul>
  <li><strong>SOLTEC.SPOS.Monitor</strong>: Proyecto para la actualización de versiones.</li>
  <li><strong>SOLTEC.SPOS.Orquestador</strong>: Sincroniza SPOS al centralizado de SOLTEC 2.</li>
  <li><strong>SOLTEC.SPOS.Updater</strong>: Automatiza la actualización de nuevas versiones.</li>
  <li><strong>SOLTEC.SPOS.ServicioMonitor</strong>: Servicio de Windows que garantiza que siempre estén en ejecución las actualizaciones y sincronizaciones.</li>
</ul>
